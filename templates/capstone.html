Capstone Project

AI Audio/Video Transcription System

An advanced, end-to-end Machine Learning pipeline designed to extract, process, and transcribe audio from video files with high precision. Built to bridge the gap between multimedia content and accessible text data, this system processes complex auditory environments to deliver accurate, timestamped transcriptions.

System Architecture & Model Details

Core Architecture

Utilizes a Transformer-based sequence-to-sequence model optimized for Automatic Speech Recognition (ASR). The system isolates audio tracks from video files via FFmpeg integrations before feeding the waveform into the neural network.

  • Multi-head self-attention mechanisms
  • Robust noise-cancellation preprocessing
  • Context-aware language modeling

Performance Metrics

Trained and fine-tuned on diverse acoustic datasets to handle various accents, background noise, and speech speeds, ensuring enterprise-grade reliability for transcription tasks.

  • Word Error Rate (WER): < 5%
  • Real-Time Factor (RTF): 0.2x
  • Punctuation Accuracy: 92%
